# 农家小炒肉（鸡蛋干版本）

## 配料

- 五花肉片（河南双汇、牧原肉食、北京二商大红门）
- 瘦肉片（河南双汇、牧原肉食、北京二商大红门）
- 螺丝椒（润苗（福建）贸易、安徽源海、合肥元良、东莞市新鲜福、蜀海深圳、蜀海北京）
- 鸡蛋干（四川众润、沈师傅）
- [小炒肉调料](/配料/小炒肉调料.md)（聚慧（重庆）、烟台欣和）
- 盐
- 蒜子
- 大豆油

## 步骤

- 1. 将200g肥肉片用60g大豆油煸炒至油脂渗出；
- 2. 下入60g蒜子炒香，再下入800g瘦肉片炒至肉片变色；
- 3. 下入150g小炒肉调料，炒至肉片均匀上色，盛出暂存；
- 4. 将60g大豆油烧热，加入940g螺丝椒和6g盐，翻炒至辣椒断生；
- 5. 下入100g鸡蛋干和煸炒好的肉片，大火翻炒出锅。
