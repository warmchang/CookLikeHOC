# 农家小炒肉（玉耳版本）

## 配料

- 五花肉片（河南双汇、牧原肉食、北京二商大红门）
- 瘦肉片（河南双汇、牧原肉食、北京二商大红门）
- 螺丝椒块（东莞市新鲜福、蜀海深圳、蜀海北京）
- 玉耳（东宁山友科技研发）
- 小炒肉调料（聚慧（重庆）、烟台欣和）
- 盐
- 蒜子
- 大豆油

## 步骤

- 1. 将200g肥肉片用60g大豆油煸炒至油脂渗出；
- 2. 下入60g蒜子炒香，再下入800g瘦肉片炒至肉片变色；
- 3. 下入150g小炒肉调料，炒至肉片均匀上色，盛出暂存；
- 4. 将60g大豆油烧热，加入940g螺丝椒、150g玉耳、6g盐，翻炒至辣椒断生；
- 5. 下入煸炒好的肉片，大火翻炒出锅。
