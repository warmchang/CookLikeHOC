# 番茄菌菇肉丸

## 配料

- 肉丸（淮安拾分味道、江苏淮安苏食肉品）
- 菌菇（云南鸿贵、广运农业江苏）
- 番茄汤膏（聚慧重庆、四川新雅轩）
- 西红柿（产地：四川、山东）

## 步骤

- 1. 汤底调配：750g开水与125g番茄汤膏，搅拌均匀；
- 2. 将100g西红柿块放入餐具底部、再放上5个肉丸；
- 3. 将15g菌菇平铺在上，并加入80g番茄汤底；
- 4. 待蒸柜上汽后，蒸制6分钟。
